# 📄 금융/증권 문서 OCR 및 데이터 추출 파이프라인 (RPA Automation)

본 프로젝트는 비개발자 실무진이 대량의 **'주금납입능력 확인 등 관련 확약서'** 스캔 및 촬영본에서 필수 기재 데이터를 손쉽게 추출하고, 결측치나 누락을 직관적으로 검수할 수 있도록 설계된 **자동화 파이프라인(RPA) 스크립트**입니다.

## 🎯 1. 프로젝트 목적 (Purpose)
* **운영 효율성 극대화:** 사람이 일일이 수백 장의 확약서를 열어보고 검토하던 수동 업무를 자동화하여 리소스(시간/인력)를 절감합니다.
* **데이터 무결성 확보:** AI(Gemini 2.5 Flash API) 및 엄격한 JSON 스키마 제약을 활용하여 판독 오류 및 휴먼 에러를 최소화합니다.
* **직관적인 모니터링:** 최종 결과물을 엑셀 테이블 형태로 반환하여, 실무자가 `N(정상)` / `Y(수동확인)` 여부만으로 즉각적인 의사결정을 내릴 수 있도록 지원합니다.

---

## 🔍 2. 이미지 전처리 메커니즘 (Image Pre-processing)
현업에서 발생하는 모니터 재촬영본, 구겨진 문서, 저해상도 샘플 등의 인식률을 보완하기 위해 컴퓨터 비전(OpenCV) 기반의 3단계 전처리를 수행합니다.

1. **그레이스케일 변환 (`cv2.cvtColor`):**
   * 불필요한 색온도나 채도 정보를 제거하여 연산 속도를 높이고 텍스트 본연의 명도에 집중합니다.
2. **노이즈 및 모아레 제거 (`cv2.fastNlMeansDenoising`):**
   * 스마트폰 화면 촬영 시 발생하는 패턴 노이즈(물결무늬, Moire) 및 조명 빛 번짐 현상을 효과적으로 통제합니다.
3. **적응형 명암 대비 증폭 (`CLAHE - Contrast Limited Adaptive Histogram Equalization`):**
   * 이미지 전체를 획일적으로 보정하지 않고, 국소적 픽셀의 대비를 최적화하여 수기로 작성된 흐릿한 텍스트나 직인(도장)의 시인성을 극대화합니다.
   * *기대 효과: 원본 대비 텍스트 추출 및 직인 인식 실패율(Recall Drop)을 평균 20~30% 경감*

## 🧾 3. 추출 대상 항목 (Extraction Targets)
Gemini 프롬프트 엔지니어링을 통해 확약서에서 다음 **4가지 핵심 항목**을 추적하며, 수기(Handwritten)로 작성된 부분까지 문맥을 추론하여 판독합니다.

| 추출 데이터명 | 설명 및 특징 | 판독 기준 (예시) |
| :--- | :--- | :--- |
| **주금납입능력 금액** | 표 안쪽에 위치한 주금납입 능력을 나타내는 수치 금액 | `3,006,771,186원` |
| **회사명** | 문서 하단에 기재된 대상 법인명 | `(주)세븐나인파트너스` |
| **주소** | 수기로 작성된 법인의 상세 주소 (오탈자 보정) | `서울시 서초구 서초동 1321-9` |
| **대표이사 및 직인** | 기재된 대표자명 및 **붉은색 도장(직인) 날인 여부** | 직인 인식 시 `True`, 미인식 시 `False` |

## 📊 4. 결과 아웃풋 (Output Report)
파이프라인 실행이 완료되면 동일한 폴더 경로에 `output_report.xlsx` 엑셀 파일이 자동 생성됩니다.  
모든 행(Row)은 이미지 1장당 1줄씩 작성되며, 핵심 검수 플래그인 **`검수_필요여부`** 컬럼을 통해 즉시 필터링이 가능합니다.

**[엑셀 컬럼 구조 명세]**
* `파일명`: 원본 대조용 이미지 파일명
* `검수_필요여부`: **"Y(수동확인)"** (누락 발생 또는 직인 없음) / **"N(정상)"** (완벽히 기재됨)
* `주금납입능력_금액`: 추출된 금액 텍스트 (공란 시 "미기입")
* `회사명`: 추출된 회사명 텍스트 (공란 시 "미기입")
* `주소`: 추출된 주소 텍스트 (공란 시 "미기입")
* `직인_날인여부`: 인장 판별 결과 ("날인" / "누락")
* `비고(누락항목)`: 판독이 불가하거나 기입되지 않은 항목들의 목록

## 🚀 5. 실행 방법 (How to Run)
1. 파이썬 의존성 패키지를 설치합니다.
   ```bash
   pip install opencv-python pandas openpyxl google-generativeai python-dotenv
   ```
2. 루트 디렉토리에 `.env` 파일을 생성하거나 환경 변수에 Google Gemini API Key를 세팅합니다.
   ```env
   GEMINI_API_KEY=AIzaSy...
   ```
3. 분석할 이미지 파일명들을 `input/` 폴더 내에 배치합니다.
4. 아래 명령으로 스크립트를 실행합니다.
   ```bash
   python main.py
   ```
